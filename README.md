A simple and efficient Habit Tracker application that helps users build and maintain daily habits. Users can add, view, and delete habits, with backend support for storing and managing data.
Features:
➕ Add new habits
📋 View all habits
❌ Delete habits
💾 Backend data storage
⚡ Simple and user-friendly interface
Tech Stack:
Frontend:
HTML
CSS
JavaScript
Backend:
Node.js
Express.js
